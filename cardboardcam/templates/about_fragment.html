About

Cardboard Camera Toolkit splits (or joins) stereo 360 degree panoramas made by Google's Cardboard Camera. Given a photo taken with Cardboard Camera, it can split it into a stereo pair and an audio file. Given a pair of panoramic images and an optional audio file, it can produce .vr.jpg format JPEG images with XMP data suitable for viewing in Cardboard Camera.

The source code is here: https://bitbucket.org/pansapiens/cardboardcam. Feel free to fork it/send bug reports/pull requests.
